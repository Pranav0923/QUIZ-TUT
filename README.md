# 🎯 QUIZ-TUT

An interactive and dynamic Quiz Application built using HTML, CSS, and JavaScript. Test your knowledge with a smooth UI, instant feedback, and score tracking.

---

## 🚀 Live Demo

👉 https://Pranav0923.github.io/QUIZ-TUT/

---

## 📌 Features

✨ Interactive quiz interface
⏱️ Timer-based questions
📊 Score calculation & result display
🔁 Restart quiz functionality
📱 Responsive design (works on mobile & desktop)

---

## 🛠️ Tech Stack

* 🌐 HTML5
* 🎨 CSS3
* ⚡ JavaScript (Vanilla JS)

1. Clone the repository

```

```

2. Open `index.html` in your browser
